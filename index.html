Projects

Here are a few projects I have worked on since joining university.

  • Simulation and Diagram Maker Program of Electric Circuits
    Course: Object-Oriented Programming

    Our project is a SPICE-inspired simulator written in Java. The program takes a text file which contains all the circuit nodes and components labeled according to specific rules. By using Nodal Analysis, the program solves the circuit for each time step and sets a voltage value for each node. Our program can simulate circuits including elements such as resistor, capacitor, inductor, voltage source (DC and Sinusoidal), current source (DC and Sinusoidal), and linear dependent sources. Other than simulating, the program is able to draw a circuit based on the input text file.

    1. DC Operating Point Analysis
    2. Transient Analysis
    3. Drawing Circuit Diagram

  • MIPS Assembly Processor Unit
    Course: Computer Architecture and Design

    We implemented Verilog HDL code for single cycle and pipelined MIPS processor which is based on 32-bit MIPS instruction set. In this project, the HDL code was partitioned into several sections including the datapath, controller, data memory and instruction set. The 5-stage pipeline CPU implementation was significantly faster in performance since, theoretically, in each clock cycle 5 instructions were handled. The main challenge in pipeline implementation, however, was data forwarding and branch hazard.

  • Ethernet Data Link Layer Network
    Course: Logic Circuits & Lab

    In this project, we implemented Ethernet frame receiver in Verilog which is a data unit on an Ethernet link that transports an Ethernet frame as its payload. The frame characteristics include physical layer and data link layer. The data link layer itself consists of three consecutive parts: Header, Payload and Frame check sequence. Depending on the Payload size, we may receive various Header parts. In addition, the FCS is dedicated to detect possible errors of a communicated frame. To achieve that, we implemented generic CRC32 in Verilog which computes the frame check sequence for the received frame based on a generator polynomial.
